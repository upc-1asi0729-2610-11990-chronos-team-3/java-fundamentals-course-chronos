# 🎓 Fundamentos de Java para Principiantes

## 📘 Resumen del Curso

Este curso de aproximadamente 1 hora introduce a estudiantes de secundaria a los fundamentos de la programación con Java, incluyendo conceptos básicos de Programación Orientada a Objetos (POO).

El contenido está diseñado para ser claro, visual e interactivo, utilizando ejemplos prácticos y herramientas 100% en línea.

**¡No requiere descargas!** Solo necesitas un navegador web.

- ⏱ **Duración total**: ~60 minutos
- 👨‍🎓 **Público objetivo**: Estudiantes de 12 a 17 años sin experiencia previa
- 📚 **Prerrequisitos**: Ninguno
- 🌐 **Herramientas necesarias**: Navegador web (Chrome, Firefox, Edge, Safari)

## 🧠 Recomendaciones para el estudiante

- Pausa los videos para practicar.
- Modifica el código para experimentar.
- No temas equivocarte: los errores son parte del aprendizaje.

---

## ⚠️ Errores comunes a evitar

- Olvidar el punto y coma (`;`)
- No inicializar variables
- Confundir `=` con `==`
- No cerrar correctamente llaves `{}`

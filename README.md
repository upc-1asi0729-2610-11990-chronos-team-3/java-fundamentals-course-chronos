# 🎓 Fundamentos de Java para Principiantes

## 📘 Resumen del Curso

Este curso de aproximadamente 1 hora introduce a estudiantes de secundaria a los fundamentos de la programación con Java, incluyendo conceptos básicos de Programación Orientada a Objetos (POO).

El contenido está diseñado para ser claro, visual e interactivo, utilizando ejemplos prácticos y herramientas 100% en línea.

**¡No requiere descargas!** Solo necesitas un navegador web.

- ⏱ **Duración total**: ~60 minutos
- 👨‍🎓 **Público objetivo**: Estudiantes de 12 a 17 años sin experiencia previa
- 📚 **Prerrequisitos**: Ninguno
- 🌐 **Herramientas necesarias**: Navegador web (Chrome, Firefox, Edge, Safari)

## 🎥 Secuencia de Lecciones

### 📗 Lección 1: ¿Qué es la programación y Java? (12 minutos)

- **Encargado**: Josep Eliu Melgarejo Quiroz
- **Descripción**: Introducción a la programación y al lenguaje Java. Se crea el primer programa "Hola Mundo".
- 🎬 **Video**: https://www.youtube.com/watch?v=Lsf60_-uyFs&list=PLKaKd42m03VQ&index=1
- **Conclusiones clave**:
  - Explicar qué significa programar.
  - Identificar la entrada, el proceso y la salida de un programa.
  - Definir qué es un algoritmo.
  - Diferenciar un algoritmo del código fuente.
  - Reconocer las etapas básicas para resolver un problema mediante programación.
  - Explicar qué es un lenguaje de programación.
  - Describir las características generales de Java.
  - Reconocer la función del compilador javac.
  - Diferenciar el código fuente del bytecode.

---

- **Encargado**: [Integrante]

- 🎬 **Video**: [LINK_YOUTUBE_LECCION_2]

### 📙 Lección 3: Entrada de datos y estructuras de control (12 minutos)

- **Encargado**: Flores Eusebio, Angel Thyago
- **Descripción**: Programas interactivos con entrada de datos, decisiones y repetición.
- 🎬 **Video**: https://youtu.be/l1yXSllyxOw
- 💻 **Práctica**: https://www.jdoodle.com/ga/seUMGnzzPnatzOEVXCHLrQ%3D%3D
- **Conclusiones clave**:
  - Uso de `Scanner` para entrada de datos.
  - Condicionales `if-else`.
  - Bucles `for` y `while`.

---

### 📕 Lección 4: Métodos e Introducción a la POO (12 minutos)

- **Encargado**:  Josep Eliu Melgarejo Quiroz
- **Descripción**: Creación de métodos (funciones) y primeros conceptos de clases y objetos.
- 🎬 **Video**: https://www.youtube.com/watch?v=UvI1Yse6Eos&list=PLKaKd42m03VQ&index=3
- 💻 **Práctica**: https://www.jdoodle.com/ga/iIVfQGCCDuXx%2BcBLUCqInQ%3D%3D
- **Conclusiones clave**:
  - Diferenciar una clase de un objeto.
  - Reconocer atributos y métodos dentro de una clase.
  - Explicar la función de un método.
  - Identificar el tipo de retorno, el nombre, los parámetros y el cuerpo de un método.
  - Crear y llamar métodos mediante un objeto.
  - Utilizar parámetros para enviar información a un método.
  - Diferenciar un método void de uno que devuelve un valor.
  - Utilizar return para devolver un resultado.

---

### 📒 Lección 5: POO, encapsulación y proyecto final (25:29 minutos)

- **Encargado**: Ayala Fernandez, Jorge Brayan
- **Descripción**: Aplicación de POO mediante encapsulación y desarrollo de un programa completo.
- 🎬 **Video**: https://youtu.be/8VpIUphIWqM
- 💻 **Proyecto final**: https://www.jdoodle.com/ga/358IQwMTxDgbiuQaQirRsA%3D%3D
- **Conclusiones clave**:
  - Uso de `private`, getters y setters.
  - Creación de objetos.
  - Desarrollo de programa: **Registro de Estudiante**.

## 🚀 Proyecto Final

Se desarrolla una aplicación básica en Java que permite:

- Crear un objeto tipo **Estudiante**
- Asignar atributos como nombre y edad
- Aplicar encapsulación
- Mostrar información en consola

## 🧠 Recomendaciones para el estudiante

- Pausa los videos para practicar.
- Modifica el código para experimentar.
- No temas equivocarte: los errores son parte del aprendizaje.

---

## ⚠️ Errores comunes a evitar

- Olvidar el punto y coma (`;`)
- No inicializar variables
- Confundir `=` con `==`
- No cerrar correctamente llaves `{}`
